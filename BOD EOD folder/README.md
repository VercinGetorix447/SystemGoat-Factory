# Docs Folder Structure

## /docs/mcps/
MCP server documentation, configs, and inventory.
Drop MCP-related docs here.

## /docs/guides/
Tutorials, API docs, framework references.
Add any docs you want Claude to reference during work.

## /docs/project/
Project-specific notes, requirements, architecture decisions.
BuildRightPros specs, SystemGoat plans, etc.

## /mcp-backups/
Automatic backup copies of MCP configurations.
